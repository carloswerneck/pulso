Documentação da API

Todas as rotas /api/* exigem o header x-api-key com a chave definida na variável de ambiente API_KEY. 

x-api-key: SUA_API_KEY

POST /api/medicoes

Registra uma nova medição. Usado pelo n8n e pelo formulário web.

Campos

CampoTipoObrigatórioDescrição
sistolicaintegersim60–260
diastolicaintegersim30–180
bpmintegernão30–220
medido_emstringnãoISO 8601 (default: agora)
observacaostringnãoTexto livre
origemstringnãomanual ou n8n (default: manual)

curl

curl -X POST https://pulso.whatchat.app.br/api/medicoes \
  -H "Content-Type: application/json" \
  -H "x-api-key: SUA_API_KEY" \
  -d '{
    "sistolica": 128,
    "diastolica": 82,
    "bpm": 76,
    "medido_em": "2026-06-17T08:30:00",
    "observacao": "Antes do cafe da manha",
    "origem": "n8n"
  }'

Resposta 201

{
  "id": 42,
  "sistolica": 128,
  "diastolica": 82,
  "bpm": 76,
  "medido_em": "2026-06-17T08:30:00",
  "observacao": "Antes do cafe da manha",
  "origem": "n8n",
  "classificacao": "estagio1",
  "alerta_crise": false
}

GET /api/medicoes?periodo=7d|30d|1a

Lista medições do período em ordem cronológica.

curl

curl "https://pulso.whatchat.app.br/api/medicoes?periodo=30d" \
  -H "x-api-key: SUA_API_KEY"

Resposta 200

{
  "periodo": "30d",
  "medicoes": [
    {
      "id": 40,
      "sistolica": 130,
      "diastolica": 84,
      "bpm": 72,
      "medido_em": "2026-05-20T07:15:00",
      "observacao": "Em repouso",
      "origem": "manual",
      "classificacao": "estagio1"
    }
  ]
}

GET /api/medicoes/resumo?periodo=7d|30d|1a

Retorna médias e última medição do período. alerta_crise: true se a última medição for crise hipertensiva.

curl

curl "https://pulso.whatchat.app.br/api/medicoes/resumo?periodo=30d" \
  -H "x-api-key: SUA_API_KEY"

Resposta 200

{
  "periodo": "30d",
  "media_sistolica": 132,
  "media_diastolica": 85,
  "media_bpm": 76,
  "total_medicoes": 24,
  "ultima_medicao": {
    "sistolica": 128,
    "diastolica": 82,
    "bpm": 76,
    "medido_em": "2026-06-17T08:30:00",
    "classificacao": "estagio1"
  },
  "alerta_crise": false
}

GET /api/medicoes/pdf?periodo=7d|30d|1a

Gera e retorna um PDF com o relatório do período.

curl

curl "https://pulso.whatchat.app.br/api/medicoes/pdf?periodo=30d" \
  -H "x-api-key: SUA_API_KEY" \
  -o relatorio.pdf

DELETE /api/medicoes/:id

Remove uma medição pelo ID.

curl

curl -X DELETE https://pulso.whatchat.app.br/api/medicoes/42 \
  -H "x-api-key: SUA_API_KEY"

Resposta 200

{ "ok": true }
